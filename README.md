# RCJ-Soccer-AGSG

Software used by my team (AGSG) in Portuguese Robotics Open 2015 in Soccer Secondary Light Weight League, where we got 2nd Place in Individual category.

# Hardware

## Striker

 Processing Units

* 1x Picaxe 40x1
* 1x Picaxe 20x2
* 1x Picaxe 18x1

Sensors

* 3x SRF04
* 1x SRF02
* 1x CMPS03
* 3x Line Sensor
* 8x IR Sensor

Actuators

* 4x Motors
* 1x Dribbler

## Goalkeeper

 Processing Units

* 1x Picaxe 40x1
* 2x Picaxe 20x2

Sensors

* 4x SRF04
* 1x CMPS03
* 3x Line Sensor
* 8x IR Sensor

Actuators

* 4x Motors
